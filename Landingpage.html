Summary: Latitude vs. X




Equator is the great circle around the Earth that is everywhere equidistant from the geographic poles and lies in a plane perpendicular to the Earth’s axis. This geographic, or terrestrial, Equator divides the Earth into the Northern and Southern hemispheres and forms the imaginary reference line on the Earth’s surface from which latitude is reckoned; in other words, it is the line with 0° latitude.We conducted a project which goal was to analyze how weather changes as you get closer to the equator. In order to do so,we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.


Once the dataset was assembled, Matplotlib was used to plot various aspects of weather vs. latitude. Factors analyzed included: temperature, cloudiness, wind-speed and humidity.


Moreover, Matplotlib was used in order to generate plots aspects of the weather based on latitude. The goal was to visualize a world map using weather plots. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
